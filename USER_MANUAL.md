## Getting Started

### Creating an Account

1. **Visit the Home Page**
   - Navigate to the Guestify homepage
   - Click "Sign Up" in the top right corner

2. **Choose Your Role**
   - Select "Book Experiences" if you want to be a Guest
   - Select "Host Listings" if you want to be a Host

3. **Fill in Your Information**
   - Enter your full name
   - Provide your email address
   - Enter a phone number (optional)
   - Create a password (must contain uppercase, lowercase, and number)
   - Confirm your password

4. **Verify Your Email**
   - Check your email inbox
   - Click the verification link
   - Return to the app and sign in

### Signing In

1. Click "Sign In" on the homepage
2. Enter your email and password
3. Click "Sign In"
4. You'll be redirected to your dashboard based on your role

### Forgot Password

1. Click "Forgot password?" on the login page
2. Enter your email address
3. Check your email for reset instructions
4. Follow the link to reset your password

## For Guests

### Searching for Listings

1. **Navigate to Search**
   - Click "Explore" in the navigation bar
   - Or go to your Guest Dashboard

2. **Apply Filters**
   - Enter search keywords
   - Select location (city, country)
   - Choose check-in and check-out dates
   - Select number of guests
   - Filter by category (Home, Experience, Service)
   - Set price range (optional)

3. **View Results**
   - Browse listing cards
   - Click any listing to view details

### Viewing Listing Details

1. **Listing Information**
   - View all images in the gallery
   - Read description and amenities
   - Check location and map
   - Read reviews from other guests

2. **Making a Booking**
   - Select check-in date
   - Select check-out date
   - Choose number of guests
   - Click "Reserve" button

### Completing Payment

1. **Review Booking Summary**
   - Verify listing details
   - Check dates and guests
   - Review price breakdown

2. **Select Payment Method**
   - Credit/Debit Card: Enter card details
   - E-Wallet: Select wallet (PayPal, Stripe, etc.)
   - Bank Transfer: Get bank details

3. **Complete Payment**
   - Review total amount
   - Click "Confirm and Pay"
   - Wait for confirmation
   - You'll be redirected to your bookings

### Managing Bookings

1. **View Your Bookings**
   - Go to "My Bookings" from dashboard
   - Filter by status (All, Upcoming, Completed)

2. **Booking Actions**
   - View listing details
   - Message the host
   - Cancel booking (if allowed)

### Wishlist

1. **Add to Wishlist**
   - Click heart icon on listing
   - View saved listings in "Wishlist"

2. **Manage Wishlist**
   - Remove items
   - Book from wishlist

## For Hosts

### Creating a Listing

1. **Navigate to Listings**
   - Go to Host Dashboard
   - Click "Listings" in sidebar
   - Click "Create New Listing"

2. **Fill in Basic Information**
   - Enter listing title (min 5 characters)
   - Select category (Home, Experience, Service)
   - Enter type (e.g., "Apartment", "Cooking Class")
   - Write detailed description (min 20 characters)

3. **Add Location**
   - Enter street address
   - City, State, Country
   - ZIP/Postal code

4. **Set Pricing**
   - Base rate per night/day
   - Discount percentage (optional)
   - Cleaning fee (optional)
   - Minimum stay requirements
   - Maximum guests

5. **Upload Images**
   - Click "Add Images"
   - Select up to 10 images (max 10MB each)
   - Images are automatically compressed
   - Remove images if needed

6. **Configure Rules & Availability**
   - Enable instant booking (optional)
   - Set cancellation policy
   - Configure check-in/check-out times
   - Set house rules

7. **Save or Publish**
   - Click "Save as Draft" to save for later
   - Click "Publish Listing" to make it live

### Managing Listings

1. **View All Listings**
   - Go to "Listings" page
   - Filter by status (All, Published, Drafts)

2. **Listing Actions**
   - **View**: See listing details
   - **Edit**: Update listing information
   - **Publish**: Make draft listing live
   - **Delete**: Remove listing (with confirmation)

### Managing Bookings

1. **View Bookings**
   - Go to Host Dashboard
   - See upcoming bookings
   - View booking calendar

2. **Booking Actions**
   - Confirm bookings
   - Message guests
   - Manage availability

### Calendar Management

1. **Set Availability**
   - Go to Calendar page
   - Mark dates as available/unavailable
   - Set special pricing for dates
   - Block dates for maintenance

## Troubleshooting

### Common Issues

#### Can't Sign In
- **Problem**: Forgot password
- **Solution**: Use "Forgot password?" link to reset

#### Email Not Verified
- **Problem**: Can't access features
- **Solution**: Check email and click verification link

#### Image Upload Fails
- **Problem**: Image too large or wrong format
- **Solution**: Use JPEG/PNG/WebP, max 10MB per image

#### Payment Fails
- **Problem**: Payment processing error
- **Solution**: Check payment method details, try again

#### Can't Find Listings
- **Problem**: No search results
- **Solution**: Adjust filters, try different dates/location
